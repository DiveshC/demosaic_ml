# Demosaicing Dataset Readme

The input Bayer pattern mosaic images can be found in `input` folder;
the corresponding reconstructed images by the naive demosaicing method
and the Matlab builtin `demosaic(...)` function are in `output`
folder.  In `rgb` folder are the ground truth RGB images.

Please note that the mosaic images are sampled using the `rggb` Bayer
pattern, meaning the upper-left corner of each image is a red pixel as
follows.
```
---------------------------------
|  Red  | Green |  Red  | Green |
|-------|-------|-------|-------|
| Green | Blue  | Green | Blue  |
|-------|-------|-------|-------|
|  Red  | Green |  Red  | Green |
|-------|-------|-------|-------|
| Green | Blue  | Green | Blue  |
---------------------------------
```
